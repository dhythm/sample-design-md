Built for Developers

開発者のために設計された
直感的なAPI

お好みの言語とフレームワークで、すぐにアプリケーション開発を始められます。

Queries

柔軟なクエリ

豊富なクエリ言語で、ネストされたドキュメントや配列もシンプルに操作できます。

// ユーザーを検索
db.users.find({
  age: { $gte: 18 },
  status: "active"
}).sort({ name: 1 })
Aggregation

データパイプライン

集約パイプラインで複雑なデータ変換と分析をデータベース内で直接実行。

// 売上集計
db.orders.aggregate([
  { $match: { year: 2026 } },
  { $group: {
    _id: "$region",
    total: { $sum: "$amount" }
  }}
])
Change Streams

リアルタイム同期

Change Streams でデータの変更をリアルタイムに監視し、イベント駆動型アーキテクチャを構築。

// 変更を監視
const stream =
  db.inventory.watch();

stream.on("change", (change) => {
  console.log(change);
});
Atlas Search

全文検索

Apache Lucene ベースの検索をネイティブに統合。ファセット、オートコンプリート、ファジー検索を実現。

// テキスト検索
db.products.aggregate([
  { $search: {
    text: {
      query: "MongoDB Atlas",
      path: "description"
    }
  }}
])
