Core Features

AIエージェント開発に必要な全てを

マルチエージェント連携

複数のAIエージェントが協調して動作するオーケストレーションシステム。スーパーバイザーパターンで複雑なタスクを自動分割。

ツール統合

カスタムツールを簡単に定義・統合。Zodスキーマによる型安全なバリデーションでエージェントの能力を拡張。

メモリシステム

会話履歴と長期記憶を管理する柔軟なメモリアーキテクチャ。コンテキストウィンドウを最適化して的確な応答を実現。

VoltAgent Console

リアルタイムでエージェントの動作を監視・デバッグ。会話フロー、ツール実行、メモリ状態を可視化するダッシュボード。

TypeScript First

完全なTypeScriptサポート。型推論、自動補完、コンパイル時チェックでエージェント開発の品質を向上。

ガードレール

入出力バリデーション、コンテンツフィルタリング、レート制限。安全なエージェント運用のためのセキュリティレイヤー。

import { Agent } from "@voltagent/core";
import { VercelAIProvider } from "@voltagent/vercel-ai";
import { openai } from "@ai-sdk/openai";

const agent = new Agent({
  name: "assistant",
  description: "A helpful AI assistant",
  llm: new VercelAIProvider(),
  model: openai("gpt-4o"),
  markdown: true,
});

// Start a conversation
const response = await agent.chat(
  "Explain quantum computing"
);
