ローカルで大規模言語モデルを実行

Ollama を使えば、Llama 3、Gemma、Mistral などのオープンソースモデルをあなたのマシンで簡単に実行できます。

ダウンロード 詳しく見る
$ollama run llama3

シンプルで強力

ターミナル一つで、最先端のAIモデルをローカル環境で動かせます。セットアップは数分で完了します。

ターミナルファースト

使い慣れたターミナルから直接モデルを実行。シンプルなCLIで直感的に操作できます。

ローカル実行

すべてのデータはローカルに保持。インターネット接続不要で、プライバシーを完全に保護します。

豊富なモデル

Llama 3、Gemma、Mistral、Code Llama など、主要なオープンソースモデルをすぐに利用可能。

REST API

組み込みのREST APIで、あらゆるアプリケーションからモデルを呼び出せます。OpenAI互換形式をサポート。

カスタマイズ

Modelfileでモデルの動作を自在にカスタマイズ。システムプロンプトやパラメータを簡単に調整。

クロスプラットフォーム

macOS、Linux、Windows に対応。Apple Silicon、NVIDIA GPU、AMD GPU を自動検出して最適化。
