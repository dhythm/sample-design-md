機能

メール配信に必要な
すべてを、ひとつに

APIファーストのアプローチで、メール配信のあらゆる課題を解決。開発者が本当に必要とする機能を提供します。

トランザクションメール

パスワードリセット、注文確認、ウェルカムメールなど。高速で信頼性の高いトランザクションメールを数行のコードで送信。

React Email

Reactコンポーネントでメールテンプレートを構築。使い慣れた技術スタックで、美しいメールをデザイン。

高い到達率

DKIM、SPF、DMARCの自動設定。専用IPアドレスとレピュテーション管理で、受信トレイへの到達率を最大化。

リアルタイム分析

送信、配信、開封、クリック、バウンス。すべてのイベントをリアルタイムで追跡し、Webhookで通知。

Audiences

コンタクトリストを管理し、セグメント化されたマーケティングメールを配信。購読解除も自動で処理。

ドメイン認証

カスタムドメインの設定をワンクリックで完了。DNS設定のガイドとリアルタイム検証で、セットアップを簡素化。

SDK言語
// Next.js API Route
import { Resend } from 'resend';
import { WelcomeEmail } from './emails';

const resend = new Resend(
  process.env.RESEND_API_KEY
);

export async function POST() {
  const { data, error } = await resend.emails.send({
    from: 'Acme <noreply@acme.com>',
    to: ['user@gmail.com'],
    subject: 'ようこそ',
    react: WelcomeEmail(),
  });

  return Response.json({ data });
}
