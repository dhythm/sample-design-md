developer experience

数行のコードで
バックエンドを構築

Supabaseのクライアントライブラリを使えば、データベースの読み書き、認証、ファイルアップロードなどを簡単に実装できます。

クイックスタート 
// Supabase クライアントを初期化
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  'https://your-project.supabase.co',
  'your-anon-key'
)

// データを取得
const { data, error } = await supabase
  .from('todos')
  .select('*')
  .order('created_at', {
    ascending: false
  })
