How It Works

3ステップで統合完了

数行のコードでAIエージェントに外部ツールの操作能力を付与できます。

01

ツールを選択

必要なツールとアクションを選び、ユーザー認証を完了します。ダッシュボードからワンクリックで設定可能です。

toolset.initiate_connection(
  app="github"
)
02

ツールを取得

SDKを使ってFunction Calling対応のツール定義を取得。お使いのAIフレームワークにそのまま渡せます。

tools = toolset.get_tools(
  actions=["GITHUB_CREATE_ISSUE"]
)
03

エージェントで実行

AIエージェントがコンテキストに基づいて最適なツールを自動選択・実行。結果は構造化されて返されます。

response = toolset.handle_tool_calls(
  response=agent_response
)
