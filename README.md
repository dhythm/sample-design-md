## Project Structure

```
.
├── README.md
├── screenshots/          # 各サイトのフルページスクリーンショット
│   ├── apple.png
│   ├── stripe.png
│   └── ...
├── <site-name>/
│   ├── DESIGN.md         # デザインシステム定義（カラー、フォント、コンポーネント）
│   └── index.html        # 自己完結型のランディングページ
└── ...
```

各サイトディレクトリには以下の 2 ファイルが含まれています:

| ファイル | 説明 |
|----------|------|
| `DESIGN.md` | [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) から取得したデザインシステム定義。カラーパレット、タイポグラフィ、コンポーネント仕様、スペーシング、レスポンシブ設計などを記述 |
| `index.html` | DESIGN.md のトークンを CSS Custom Properties で実装した、外部依存のない単一 HTML ファイル（Google Fonts のみ外部読み込み） |

## Getting Started

### サンプルサイトを閲覧する

```bash
# ブラウザで直接開く
open vercel/index.html

# または簡易サーバーで配信
npx serve .
```

### スクリーンショットを撮る

[agent-browser](https://github.com/vercel-labs/agent-browser) を使って、各サイトのフルページスクリーンショットを取得できます。

```bash
# agent-browser をインストール
npm install agent-browser

# Chrome for Testing をセットアップ
npx agent-browser install

# 1 サイトのスクリーンショットを取得
npx agent-browser open "file://$(pwd)/vercel/index.html"
npx agent-browser screenshot --full ./screenshots/vercel.png

# 全サイトを一括でスクリーンショット
for dir in */; do
  [ -f "$dir/index.html" ] || continue
  name="${dir%/}"
  npx agent-browser open "file://$(pwd)/$name/index.html"
  npx agent-browser screenshot --full "./screenshots/$name.png"
done
```

### 新しいサイトを追加する

1. [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) から DESIGN.md を取得
2. 新しいディレクトリを作成し、DESIGN.md を配置
3. DESIGN.md のデザイントークンに基づいて `index.html` を作成
4. スクリーンショットを撮って `screenshots/` に保存
