Core Features

エラーの原因を瞬時に特定

コードの問題をリアルタイムで検知し、スタックトレース、ブレッドクラム、コンテキスト情報で迅速なデバッグを実現します。

🐛

Error Monitoring

スタックトレース、ブレッドクラム、タグ付きコンテキストで、エラーの根本原因を素早く特定します。重複するエラーは自動的にグループ化されます。

Learn more →
⚡

Performance Monitoring

トランザクション追跡で遅延の原因を特定。P50 / P95 / P99 のレイテンシ分布を可視化し、パフォーマンスのボトルネックを解消します。

Learn more →
🎬

Session Replay

ユーザーの操作をビデオのように再生。エラー発生時にユーザーが何をしていたかを正確に把握し、再現手順の推測を排除します。

Learn more →
📊

Release Tracking

各リリースのエラー率、クラッシュフリーセッション率を追跡。デプロイが品質に与える影響をリアルタイムで把握できます。

Learn more →
🔔

Alerts & Notifications

カスタマイズ可能なアラートルールで、重要な問題を見逃さない。Slack、PagerDuty、メールなど、あらゆるチャネルに通知を送信します。

🔍

Distributed Tracing

マイクロサービス間のリクエストフローを可視化。サービス間の依存関係を把握し、パフォーマンスのボトルネックを特定します。

Developer Experience

数行のコードですぐに始められる

お使いのフレームワークに対応した SDK をインストールし、DSN を設定するだけ。自動的にエラーをキャプチャし、ソースマップを使ったスタックトレースで問題箇所を正確に表示します。

View Documentation 
// Sentry SDK の初期化
import * as Sentry from "@sentry/react";

Sentry.init({
  dsn: "https://examplePublicKey"
      + "@o0.ingest.sentry.io/0",
  tracesSampleRate: 1.0,
  replaysSessionSampleRate: 0.1,
});

// エラーは自動的にキャプチャされます
function processPayment(order) {
  try {
    return chargeCard(order);
  } catch (err) {
    Sentry.captureException(err);
  }
}
