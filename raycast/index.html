機能

ワークフローを
加速させる

日常のタスクを効率化し、より速く、よりスマートに作業するためのツール群。

高速検索

アプリケーション、ファイル、ブックマーク、その他あらゆるものを瞬時に検索。Spotlightを超える速度と精度を実現します。

ウィンドウ管理

キーボードショートカットでウィンドウをすばやく配置。マルチディスプレイ環境でも完璧に動作します。

クリップボード履歴

コピーしたすべてのテキスト、画像、リンクにいつでもアクセス。検索とフィルタリングで必要なものを即座に見つけます。

スニペット

よく使うテキストをスニペットとして保存し、キーワードで瞬時に展開。定型文の入力時間を大幅に削減します。

Raycast AI

AIアシスタントがあなたのワークフローに統合。テキスト生成、コード解説、翻訳など、あらゆる作業をサポートします。

拡張エコシステム

コミュニティが開発した数千の拡張機能で、Jira、GitHub、Slack、Notionなどのお気に入りのツールと連携します。

プラットフォーム

開発者のために
設計されたAPI

React と TypeScript で独自の拡張機能を構築し、チームやコミュニティと共有できます。

API

直感的な拡張機能API

React コンポーネントベースのAPIで、馴染みのある開発体験を提供します。

import { List } from "@raycast/api";

export default function Command() {
  return <List>...</List>;
}
