# Replicate Design System Summary

## Color Palette

**Primary Colors:**
- Replicate Dark: #202020 (primary text/dark surface)
- Replicate Red: #ea2804 (brand core)
- Secondary Red: #dd4425 (button borders, hover states)

**Accent & Status:**
- Status Green: #2b9a66 (operational badges)
- GitHub Dark: #24292e (code blocks)

**Backgrounds:**
- Pure White: #ffffff
- Near White: #fcfcfc

**Neutrals:**
- Medium Gray: #646464
- Warm Gray: #4e4e4e
- Mid Silver: #8d8d8d
- Light Silver: #bbbbbb

**Hero Gradient:** Orange (#ea2804) → red → magenta → hot pink

## Typography

**Families:**
- Display: rb-freigeist-neue
- Body/UI: basier-square
- Code: jetbrains-mono

**Key Sizes & Weights:**
- Display Mega: 128px, weight 700
- Hero Headline: 72px, weight 700, -1.8px letter-spacing
- Section Heading: 48px, weight 400–700
- Body Large: 20px, weight 400
- Body/Button: 16–18px, weight 400–600
- Code: 14px, weight 400
- Small/Tag: 12px, weight 400 (lowercase transform)

## Layout & Spacing

- Base unit: 8px
- Section vertical spacing: 96–192px
- Button padding: varies (0px 4px to 6px 56px)
- Border radius: 9999px (pill-shaped) exclusively

## Component Styles

### Buttons

**Dark Solid:**
- Background: #202020
- Text Color: #fcfcfc
- Padding: 0px 4px
- Border: 4px solid #202020
- Border-Radius: 9999px

**White Outlined:**
- Background: #ffffff
- Text Color: #202020
- Border: 1px solid #202020
- Border-Radius: 9999px

**Transparent Glass:**
- Background: rgba(255, 255, 255, 0.1)
- Text Color: #202020
- Padding: 6px 56px 6px 28px
- Border: transparent
- Outline: 1px solid #bbbbbb
- Border-Radius: 9999px

### Cards & Containers

- Background: White or subtle gray
- Border: 1px solid #202020
- Border-Radius: 9999px
- Shadow: Minimal (relies on borders for depth)
- Accent Border Variant: 1px solid #ea2804 for featured items

### Navigation

- Background: White
- Text Color: #202020
- Link Decoration: underline dotted #bbbbbb
- CTA Button: Dark pill button styling
- Border-Radius: 9999px

### Hero Section

- Gradient: Multi-stop flowing through orange (#ea2804) → red → magenta → hot pink
- Text Color: White
- Headline: 72px rb-freigeist-neue, weight 700
- Letter-Spacing: -1.8px
- Full-width with centered content

### Status Badges

- Background: #2b9a66
- Text Color: White
- Font: 14px basier-square
- Border-Radius: 9999px

### Links

- Text Color: #202020
- Text-Decoration: underline dotted #bbbbbb

### Code Blocks

- Background: #24292e
- Text Color: White
- Font: JetBrains Mono, 14px
- Border-Radius: 9999px
- Line-Height: 1.43

### Manifesto Section

- Background: #202020
- Text Color: White
- Font: 128px rb-freigeist-neue, weight 700
- Line-Height: 1.00

## Key Characteristics

- Massive display typography combined with exclusively pill-shaped geometry
- No drop shadows — depth from borders and color only
- Dotted-underline links in Light Silver (#bbbbbb)
- Developer-focused, bold, high-contrast design
- Community-powered energy through model galleries and code examples
