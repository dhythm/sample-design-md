AIコーディングはターミナルで完結する

OpenCodeは、ターミナルネイティブのAIコーディングプラットフォームです。 コードベースを理解し、あなたのワークフローに溶け込むAIアシスタントで、 開発速度を劇的に向上させます。

OpenCodeは開発者のために設計されたAIコーディングツールです。 エディタを切り替える必要はありません。

>_

ターミナルネイティブ

GUIは不要。ターミナルで起動し、ターミナルで完結する。 あなたのシェル環境にシームレスに統合されます。

{}

コードベース理解

プロジェクト全体をインデックス化し、ファイル間の依存関係や アーキテクチャを深く理解した上でコードを生成します。

AI

マルチモデル対応

Claude、GPT-4、Geminiなど主要なLLMプロバイダーに対応。 タスクに応じて最適なモデルを選択できます。

//

自動テスト実行

コード変更後に自動でテストを実行し、結果をフィードバック。 壊れたテストはAIが修正案を提示します。

$>

Git統合

変更差分の確認、コミットメッセージの生成、PRの作成まで ターミナル内でワンストップで完結します。

~~

オープンソース

完全にオープンソース。コードを監査し、自分のニーズに合わせて カスタマイズし、コミュニティに貢献できます。

3ステップで始める

  1. 1

    インストール

    npmまたはHomebrewでワンコマンドインストール。 セットアップは30秒で完了します。

    $ npm install -g opencode
  2. 2

    プロジェクトを初期化

    プロジェクトのルートディレクトリで初期化コマンドを実行。 コードベースが自動的にインデックスされます。

    $ cd your-project && opencode init
  3. 3

    自然言語でコーディング

    やりたいことを日本語や英語で入力するだけ。 AIがコードを理解し、最適な変更を提案・実行します。

    $ opencode "認証ミドルウェアにレート制限を追加して"
