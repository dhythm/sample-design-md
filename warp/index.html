Features

Everything you need,
nothing you don't

Warp AI

自然言語でコマンドを入力。AIがコマンド候補を提案し、エラーの修正もサポートします。

Block-based UI

コマンドと出力をブロック単位で管理。選択、コピー、共有が直感的に行えます。

Rust 製の高速エンジン

Rust でネイティブに構築。起動から描画まで、あらゆる操作がミリ秒単位で完了します。

チームコラボレーション

ワークフローやコマンド履歴をチームで共有。ナレッジの属人化を防ぎます。

モダンなエディタ体験

構文ハイライト、オートコンプリート、マルチカーソル。IDE のような編集体験をターミナルで。

Workflows

よく使うコマンドをパラメータ付きで保存。チーム全体で再利用可能なワークフローに。

Warp AI

AIがターミナルを変える

コマンドを覚える必要はありません。やりたいことを自然言語で伝えるだけ。Warp AI がコンテキストを理解し、最適なコマンドを提案します。エラーの原因分析と修正も自動で行います。

Modern input

テキストエディタのように入力する

Warp の入力エリアは、従来のターミナルとは一線を画します。複数行編集、カーソル移動、コマンド履歴の検索。すべてがモダンなエディタのように動作します。
